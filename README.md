# Simple k8s backup
Simple backup YAML files from Kubernetes

Backup YAML files with lists:
Nodes, namespaces, deployments, replicasets, daemonsets, statefulsets, pods, services, serviceentry, virtualservice, endpoints, pv, pvc, sc, configmaps and secrets.

Backup YAML files with configurations:
Nodes, deployments, replicasets, daemonsets, statefulsets, pods, services, serviceentry, virtualservice, endpoints, pv, pvc, sc, configmaps and secrets.

Then the files are archived in .zip
